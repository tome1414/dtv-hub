## ⚠️ **重要：このセッションの方針**

### AI記事・大量投稿を避ける理由

```
❌ 悪い例：
  - AI が自動生成した記事を大量投稿
  - 品質・オリジナリティが低い
  - Google のコアアップデートで順位下落
  - ユーザー満足度が低い

✅ 良い例：
  - 1ヶ月に3-5本程度の質の高い記事
  - ユーザーの実際のニーズに応える
  - オリジナルな情報・経験を含む
  - 他サイトにない視点
```

### 戦略的アプローチ

```
キーワード調査（既実施）
    ↓
キーワード優先度付け（このセッション）
    ↓
関連する記事をグループ化
    ↓
各グループで「軸となる記事」を1本確定
    ↓
1ヶ月スケジュール決定
    ↓
品質チェックリスト作成
```

## 📊 実装ステップ

### Step 1：キーワード優先度付け

**ユーザーから既存キーワードリスト を受け取る**

```
例：
- DTV ビザ 申請
- DTV ビザ 必要書類
- バンコク 家賃
- DTV 月収要件
- 東京 タイ大使館
- etc...
```

**各キーワードに優先度スコアを付与**

```typescript
interface KeywordAnalysis {
  keyword: string              // 「DTV ビザ 申請」
  searchVolume: number         // 月間検索数（推定）
  difficulty: 'low' | 'mid' | 'high'  // 競合度
  intent: 'informational' | 'navigational' | 'commercial'
  priority: number             // 優先度スコア（1-10）
  recommendedOrder: number     // 投稿順序
  relatedKeywords: string[]    // 関連キーワード
  notes: string               // 備考
}
```

**優先度計算式**
```
Priority = (SearchVolume × 0.4) + ((10 - Difficulty) × 0.4) + (Relevance × 0.2)

例：
- 「DTV ビザ 申請」：検索多い、競合中、高関連性 → Priority 8
- 「バンコク マンション 家賃 DTV」：検索少ない、競合低、高関連性 → Priority 6
```

### Step 4：記事テンプレート・ガイドライン

**ユーザーが意識すべき品質基準**

```markdown
# 記事テンプレート：DTV関連ガイド

## メタデータ
- メタタイトル（60文字以内）：「DTV ビザ申請【2026年最新版】東京大使館完全ガイド」
- メタディスクリプション（160文字以内）：「東京のタイ王国大使館でのDTV申請を...」
- ターゲットキーワード：「DTV ビザ 申請」
- 関連キーワード：「DTV 東京」「タイ 長期滞在」

## H1（1つだけ）
DTV ビザ申請【2026年最新版】東京大使館での完全ガイド

## H2 セクション（目次）
1. DTV ビザとは
2. 申請条件
3. 必要な書類
4. 申請手順（ステップバイステップ）
5. 審査期間
6. よくある質問
7. まとめ

## 内部リンク戦略
- 関連記事へのリンクを3-5個自然に挿入
- 例：「月収要件について詳しくはこちら」→ /blog/dtv-monthly-income

## 外部リンク
- 信頼性高い情報源のみ（タイ大使館公式サイトなど）
- 3-5個程度が目安

## 画像・メディア
- オリジナル画像優先
- スクリーンショットは日本語キャプション付き
- SEO用 alt テキスト必須

## 記事長
- 最小 1500 文字
- 目標 2500-3500 文字
- ただし「詰め込み」は避ける

## 品質チェックリスト
- [ ] ユーザーの実際のニーズに答えているか
- [ ] オリジナルな情報・視点があるか
- [ ] 信頼性のある情報源に基づいているか
- [ ] 正確な情報か（誤情報がないか）
- [ ] 実体験に基づいているか（できれば）
- [ ] 読みやすく構成されているか
- [ ] 内部リンクは自然か
```
