
## ── 記事構造ガイド ──

### H1（1つだけ・タイトルと一致）

```markdown
# DTV ビザとは？【2026年最新版】完全ガイド
```

**ルール：**
- タイトルとH1は完全一致
- ターゲットキーワードを含む
- 年号・「完全ガイド」「最新版」など信頼性を示すワードを入れる

---

### リード文（150〜200字）

```markdown
DTV（Destination Thailand Visa）は、デジタルノマドやリモートワーカーが
タイに長期滞在するための5年間有効なビザです。
この記事では、DTVビザの基本概念から申請条件・費用まで、
2026年の最新情報をもとに徹底解説します。
```

**ルール：**
- ターゲットキーワードを最初の100字以内に含める
- 「この記事でわかること」を明示
- 読者の悩みに共感する書き出しでもOK

---


## ── 内部リンク戦略 ──

| パターン | 例 |
|--------|-----|
| 関連キーワードへの自然な誘導 | 「申請手順の詳細は[こちらの記事](/blog/...)をご覧ください」 |
| 補足情報として | 「50万バーツの証明方法については[別記事](/blog/...)で解説しています」 |
| まとめセクションで複数リンク | 「関連記事：[...]、[...]」 |

**ルール：**
- 記事内に3〜5個の内部リンクを自然な形で挿入
- 同じリンクを2回以上使わない
- アンカーテキストにキーワードを含める

---

## ── 外部リンク ──

**使用OK：**
- タイ王国大使館公式サイト（https://site.thaiembassy.jp/）
- Thai e-Visa 公式（https://thaievisa.go.th/）
- タイ移民局公式
- 政府・公的機関のサイト

**使用NG：**
- 個人ブログ（一般）
- 情報が古いサイト
- 競合サイト

## ── 画像・メディアガイドライン ──

```
/images/blog/[記事slug]/
  ├── cover.jpg        （OGP用：1200×630px）
  ├── step-01.jpg      （手順説明画像）
  ├── step-02.jpg
  └── infographic.jpg  （まとめ図解）
```

**必須：**
- すべての画像に `alt` テキストを設定
- alt テキストにキーワードを自然に含める
- ファイル名はケバブケースで意味のある名前に

## ── 品質チェックリスト ──

公開前に以下をすべて確認してください：

### コンテンツ品質
- [ ] ターゲットキーワードが H1・リード文・最初の見出しに含まれているか
- [ ] ユーザーの検索意図（知りたいこと）に完全に答えているか
- [ ] オリジナルな情報・視点・体験が含まれているか
- [ ] 他のサイトにない情報（実体験・独自データ）があるか
- [ ] 情報は最新かつ正確か（ファクトチェック済みか）
- [ ] AI丸写しではなく、人間が推敲した文章か

### SEO
- [ ] メタタイトルは60文字以内か
- [ ] メタディスクリプションは160文字以内か
- [ ] H1 は1つだけか
- [ ] H2 → H3 の階層が正しいか（H3 の後に H2 が来ていないか）
- [ ] 内部リンクが3個以上あるか
- [ ] 画像のalt テキストが設定されているか

### 読みやすさ
- [ ] 一文が80字以内か（長すぎる文を分割）
- [ ] 段落は3〜4文程度か
- [ ] 箇条書きを活用しているか
- [ ] 太字・強調が適切に使われているか
- [ ] 目次があるか（2000字以上の記事）

### 信頼性
- [ ] 数字・データは信頼できるソースから引用しているか
- [ ] 公式サイトへの外部リンクがあるか
- [ ] 「最終更新日」が表示されているか
